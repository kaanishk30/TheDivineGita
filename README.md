# Bhagavad Gita Comic App 📚⚡

A vibrant, comic-style Next.js application for reading the Bhagavad Gita in an interactive and engaging format!

## Getting Started

1. Install dependencies:
```cmd
npm install
```

2. Run the development server:
```cmd
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## Project Structure

- `/app` - Next.js app directory with pages and layouts
- `/components` - React components
- `/data` - JSON data files for chapters and verses
- `/utils` - Utility functions
- `/public` - Static assets

## Features

- 🎨 Comic book style design with vibrant colors and bold typography
- 📖 Browse all 18 chapters of the Bhagavad Gita
- 📜 Read verses with Sanskrit text, transliteration, and word meanings
- 💫 Engaging visual design with speech bubbles and comic panels
- 📱 Responsive design for mobile and desktop
- ⚡ Fast and smooth navigation between chapters
